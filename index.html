Stock de Productos

Aplicación de escritorio desarrollada con C# en .NET FRAMEWORK.
La aplicación brinda la funcionalidad de manipular un stock de productos, pudiendo realizar lectura de los mismos como registros de una lista y mostrando diferentes datos. Además, contiene la funcionalidad de creación, modificación y eliminación de artículos a través de diferentes ventanas creadas en el framework .NET.
La persistencia de datos se realiza a través de una base de datos SQL (SQL SERVER), utilizando el conjunto de clases ADO.NET.
